# 企业招采平台-littlet 登录页项目说明书

你好！这是为你准备的登录页项目文档。在这个项目中，我们还原了“企业招标采购平台”的登录界面，并实现了基础的交互逻辑。

## 1. 项目结构
- **index.html**: 网页主入口，包含账号、CA、手机号三种登录方式。
- **help.html**: 驱动下载与帮助中心页面。
- **reset.html**: 密码重置与账号申诉页面。
- **css/style.css**: 全局样式表。
- **js/**: 交互逻辑代码。

## 2. 功能说明
1. **多方式登录**:
   - **账号登录**: 传统的账号密码模式。
   - **CA登录**: 切换展示专属二维码，模拟Key登录场景。
   - **手机登录**: 支持输入手机号，点击发送验证码（含60秒倒计时模拟）。
2. **界面交互**: 左蓝右白的经典布局，Tab切换流畅。
3. **辅助页面**:
   - 点击“驱动下载”跳转至下载列表页。
   - 点击“忘记密码”跳转至申诉表单页。

## 3. 使用方法
1. 直接在浏览器打开 `index.html` 即可看到效果。
2. 手机登录测试：输入任意手机号，点击“获取验证码”，按钮会进入倒计时状态。
